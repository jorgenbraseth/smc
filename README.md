# Shape modifier 2000
**The sexiest shape modifier known to man**

## Quick start
NOTE: only tested with Node 9.4.0 + NPM 6.4.0 and a Chrome browser.
### 1. Build and Run backend
    cd backend
    npm install
    npm run start
    
### 2. Build and Run frontend
    cd frontend
    npm install
    npm run start
    
### 3. Open browser
    http://localhost:3000
